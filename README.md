# Ejemplo de creación de proyecto en Django - School Management

En este repositorio crearemos un proyecto de Django paso a paso que muestre los estudiantes de distintas escuelas de una universidad. Es un proyecto simple que permite acercarse a esta tecnología.

Aquí tocaremos tema de Bases de datos, MVC, entre otros.

Toda la infomación de este tutorial se encuentra en la [Wiki](https://github.com/Manco312/SchoolManagement---Ejemplo-de-Django/wiki)
